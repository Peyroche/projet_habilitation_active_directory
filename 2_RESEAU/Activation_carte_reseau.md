# ACTIVATION CARTE RESEAU

---

## Procédure :

Pour activer la carte réseau, la procédure est la suivante :

1. Ouvrir VirtualBox

2. Choisi la machine

3. Clique sur onglet Réseau 

4. Coche la case "Activer l’interface réseau" 

5. Vérifie que : 

- Mode d’accès réseau = Réseau interne 

- Nom = le même que sur le serveur (ex : intnet) 

- Type d’interface = Intel PRO/1000 MT Desktop (82540EM)

- Câble branché = coché ✅
